# Codex Project Instructions

Before doing project work in this repository, read the root `CLAUDE.md`.

Treat `CLAUDE.md` as the source of truth for:

- project background and goals
- architecture and important workflows
- build, test, and run commands
- coding conventions and repository-specific constraints
- operational notes that affect implementation choices

After reading it, follow those instructions unless they conflict with a direct user request or higher-priority system/developer instructions in the current session.

If `CLAUDE.md` is large, read the sections relevant to the current task first, then return to it as needed instead of guessing project context.
